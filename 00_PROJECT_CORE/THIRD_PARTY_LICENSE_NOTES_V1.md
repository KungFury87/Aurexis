## Runtime-Optional Third-Party Dependency

### Pillow (PIL)

- **Where used:** `src/aurexis_lang/raster_law_bridge_v1.py` — the `_decode_png_to_rgb()` helper attempts `from PIL import Image` inside a `try` block and falls back to a hand-rolled stdlib PNG decoder on `ImportError`.
- **How used:** Public API only — `PIL.Image.open(...)`, `.convert('RGB')`, `.size`, `.getpixel(...)`. No Pillow source code is shipped, modified, or redistributed.
- **License:** Pillow is distributed under the HPND license (historically-permissive-notice-and-disclaimer). Users who install Pillow do so independently.
- **Attribution:** Usage is through the library's public API. No attribution is required for API calls. This note documents the relationship honestly regardless.
- **Link:** https://pillow.readthedocs.io

## Test-Time Optional Third-Party Dependency

### pytest

- **Where used:** The `tests/` directory contains `pytest`-style test modules. If `pytest` is installed on the host, users may run `pytest tests/ -q`.
- **How used:** Public API only (`import pytest`, `@pytest.fixture`, `@pytest.mark.parametrize`, `pytest.raises`). No pytest source code is shipped, modified, or redistributed.
- **Fallback:** The release zip ships `run_pytest_surface.py`, an authored lightweight pytest-compatible runner that supports the subset of pytest features the test suite uses. This allows the full 327-test suite to run without any third-party install.
- **License:** pytest is distributed under the MIT License.
- **Attribution:** Usage is through the library's public API. Documented here honestly.
- **Link:** https://docs.pytest.org
